Bash 用户级自定义配置

Bash 的用户级别自定义配置存放在 ~/.bashrc 或者 ~/.bash_profile 文件下；
如需添加一个全局变量：TEST=“Hello world !”

wlz@ubuntu:~$ vi ~/.bashrc
TEST="Hello world !"

测试是否添加成功？

wlz@ubuntu:~$ echo $TEST

输出空，需要告知 shell 重新加载 .bashrc 文件,执行source命令

wlz@ubuntu:~$ source ~/.bashrc
wlz@ubuntu:~$ echo $TEST
Hello world !

是为记。
