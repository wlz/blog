Vim 添加一键调试 Scheme 功能

思路

借助display函数，执行当前行代码，并将执行结果输出。

修改 Vim 配置件.vimrc，添加如下键映射配置：

nmap <F5> 0ya(%o(display )<esc>hp0:w<cr>:silent !petite %<cr>va(ddd'':<bs>

测试

(define fact
  (lambda (n)
    (cond
      ((= n 1) 1)
      (else
	(* n (fact (- n 1)))))))

(fact 5)

光标移动到(fact 5)所在行，按下F5即可执行并显示执行结果，多方便啊:)
